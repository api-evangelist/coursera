# Coursera (coursera)

Coursera is a global online learning platform offering courses, specializations, professional certificates, and degrees. The Catalog API provides public, read-only access to courses, instructors, partners, and categories; the Affiliate API supports partner programs.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/coursera/refs/heads/main/apis.yml)

## APIs
- **Coursera Catalog API** — public, read-only REST API at `https://api.coursera.org/api/courses.v1` (21k+ courses). Docs: https://building.coursera.org/app-platform/catalog/
- **Coursera Affiliate API** — for affiliate partners.
